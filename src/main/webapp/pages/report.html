Задание №1

Какой символ является признаком конца тела программы?

Правильный ответ: Точка.

Простейшая программа на Паскале схематически может быть изображена следующим образом: 
    Program N ;                     — заголовок программы;
    Var A1 ; A2 ; ... ; Ak ;        — раздел описаний;
    Begin P1 ; P2 ; ... ; Pn end.   — раздел операторов.

    Здесь:  N – имя программы (внешний идентификатор);
            Ai – описание i-й группы переменных;
            Pj – j-й оператор.
